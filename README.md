# resumo-de-videos-ia
Agente de IA que usa a API da OpenAI para resumir videos enviados do youtube

Por conta do agente usar o YoutubeTools, classe da biblioteca/framework Agno, ele aceita apenas vídeos que sejam feitos em inglês. 

No arquivo chamado .env, estará escrito "OPENAI_API_KEY=". A sua chave API precisará ser colada após o "=" para o código funcionar.
